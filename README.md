# Dash

A developer-focused macOS menu bar app for quick access to system commands and developer shortcuts.

## Features

- **Quick Actions** - Open terminal, clear DNS cache, restart Finder, toggle hidden files, empty trash
- **System Info** - IP address, hostname, macOS version, uptime
- **Dev Tools** - Kill port by number, restart Dock, purge memory, rebuild Spotlight index
- **Clipboard Tools** - Copy current path, copy IP, generate UUID, copy hostname

## Build

```bash
swift build
```

## Run

```bash
swift run Dash
```

## Test

```bash
swift run DashTests
```

## Requirements

- macOS 13+
- Swift 6.0
